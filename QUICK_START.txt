🚀 QUICK START GUIDE
==================

Bug Bounty Automated Scanner - Get started in 3 steps!

STEP 1: Install Dependencies
-----------------------------
pip install -r requirements.txt


STEP 2: Create Your Targets File
---------------------------------
Edit targets.txt and add your bug bounty program wildcards/URLs:

Example:
  *.example.com
  *.hackerone.com
  https://api.bugcrowd.com


STEP 3: Run the Scanner
-----------------------
python bug_bounty_scanner.py -i targets.txt -o results.json


STEP 4 (Optional): Analyze Results
-----------------------------------
python analyze_results.py -i results.json -o priority_targets.txt


📖 COMMON COMMANDS
==================

# Basic scan
python bug_bounty_scanner.py -i targets.txt -o results.json

# Fast scan (high concurrency)
python bug_bounty_scanner.py -i targets.txt -o results.json -c 20 -t 5

# Careful scan (low concurrency, longer timeout)
python bug_bounty_scanner.py -i targets.txt -o results.json -c 5 -t 15

# Export to CSV for Excel
python bug_bounty_scanner.py -i targets.txt -o results.csv -f csv

# Export to readable text file
python bug_bounty_scanner.py -i targets.txt -o results.txt -f txt

# Analyze and find interesting endpoints
python analyze_results.py -i results.json

# Export high-priority targets
python analyze_results.py -i results.json -o high_priority.txt

📊 OUTPUT FORMATS
=================

JSON (default):
  - Structured data with all details
  - Best for further processing

CSV:
  - Easy to import into Excel/Google Sheets
  - Good for reporting

TXT:
  - Human-readable

🔧 TROUBLESHOOTING
==================

Scan too slow?
  → Increase concurrency: -c 20

Getting timeouts?
  → Increase timeout: -t 15
  → Decrease concurrency: -c 5

No results found?
  → Check your targets file format
  → Try increasing timeout
  → Verify DNS is working

Need help?
  → Check README.md for full documentation
  → Run with -h flag for options
